📌 PROBLÈME
-----------
La secrétaire ne recevait pas toujours la notification quand le médecin 
terminait une consultation.


✅ SOLUTION
-----------
J'ai identifié qu'il y avait DEUX endroits pour "Terminer consultation" :

1. Dashboard Médecin → Section "Patient Actuel" → Bouton "Terminer consultation"
   ✅ Déjà fonctionnel (envoie la notification via SQL)

2. Page Workflow → Bouton "Terminer consultation" (en bas)
   ❌ N'envoyait PAS de notification → ✅ CORRIGÉ !


📬 MESSAGE DE NOTIFICATION
--------------------------
La secrétaire reçoit maintenant ce message dans TOUS les cas :

  "✅ Dr. Pape Dabo a terminé la consultation avec Aissatou Ba. 
      Vous pouvez faire entrer le patient suivant."


🧪 COMMENT TESTER ?
-------------------

EN TANT QUE MÉDECIN :
1. Ouvrez la console (F12)
2. Terminez une consultation (peu importe le bouton)
3. Regardez les logs :
   - Vous devez voir : 🔵 ✅ 👨‍⚕️ 👔 📝 📬
   - Le dernier log doit être : "📬 Message: Dr. ... a terminé..."
   - PAS de ❌ rouge

EN TANT QUE SECRÉTAIRE :
1. Connectez-vous
2. Regardez l'icône de notification (cloche en haut)
3. Une pastille ROUGE avec un chiffre doit apparaître
4. Cliquez dessus
5. Lisez la notification : "✅ Dr. ... a terminé la consultation avec ..."


🚨 SI ÇA NE MARCHE PAS
----------------------

1. VÉRIFIEZ QUE LA SECRÉTAIRE EST ACTIVE :
   SQL → SELECT * FROM users WHERE role = 'secretary';
   Si actif = false → UPDATE users SET actif = true WHERE role = 'secretary';

2. REGARDEZ LES LOGS CONSOLE (F12) :
   - Si vous voyez un ❌, lisez le message d'erreur
   - Si vous voyez "données manquantes", c'est probablement la secrétaire

3. VÉRIFIEZ LA BASE DE DONNÉES :
   SQL → SELECT * FROM notifications_medecin_secretaire 
         WHERE type_notification = 'consultation_ended' 
         ORDER BY created_at DESC LIMIT 5;

4. ERREUR "Key is not present in table waiting_queue" (409 Conflict) :
   ✅ CORRIGÉ ! Le problème était l'ordre des opérations.
   Maintenant la notification est envoyée AVANT de supprimer waiting_queue.
   Vous ne devriez plus voir cette erreur.
